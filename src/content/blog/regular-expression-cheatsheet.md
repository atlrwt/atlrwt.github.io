## 前言

正则表达式（通常称为 `regex` 或 `regexp`）是一种强大的工具，常用于在文本中执行模式匹配、字符串查找和替换等操作。

本文将通过一个简明的速查表介绍常用的正则表达式及其应用示例。

## 正则表达式

| 表达式     | 描述                               | 示例                                                 |
|---------|----------------------------------|----------------------------------------------------|
| ^       | 匹配输入字符串的开始位置                     | `^abc` 表示匹配以 `abc` 开头的字符串                          |
| $       | 匹配输入字符串的结束位置                     | `abc$` 表示匹配以 `abc` 结尾的字符串                          |
| .       | 匹配除换行符（`\n` 和 `\r`）之外的任意单个字符     | `a.c` 可以匹配 `abc`、`arc`、`a3c` 等，但不可以匹配 `a\nb`       |
| *       | 前面的字符出现零次或多次                     | `a*b` 可以匹配 `b`、`ab`、`aab`、`aaab` 等                 |
| +       | 前面的字符出现一次或多次                     | `a+b` 可以匹配 `ab`、`aab`、`aaab`，但不会匹配 `b`             |
| ?       | 前面的字符出现零次或一次                     | `colou?r` 可以匹配 `color` 和 `colour`                  |
| [list]  | 用于匹配括号内的任一字符                     | `[abc]` 表示匹配 `a`、`b` 或 `c`                         |
| [^list] | `^` 在方括号中使用时表示非，即不匹配方括号内的字符      | `[^abc]`表示匹配不是 `a`，`b` 和 `c` 的任意字符                 |
| [n1-n2] | 匹配指定范围内的任意字符                     | `[A-Z]` 可以匹配任意大写字母                                 |
| {n}     | 匹配确定的 `n` 次（`n>=0`）              | `a{2}` 表示匹配 `aa`                                    
| {n, }   | 至少匹配 `n` 次（`n>=0`）               | `a{2, }` 可以匹配 `aa`、`aaa`、`aaaa` 等                  |
| {n, m\} | 最少匹配 `n` 次且最多匹配 `m` 次（`0<=n<=m`） | `a{2,4}` 可以匹配 `aa`、`aaa` 或 `aaaa`                  |
| \\      | 转义字符，用于匹配那些在正则表达式中有特殊意义的字符。      | `a\*b` 表示匹配 `a*b`                                  |
| \|      | 表示逻辑或操作                          | `gd\|good` 可以匹配 `gd` 或 `good`                      |
| ()      | 用于创建一个子表达式，这个子表达式可以被整体操作         | `g(la\|oo)d` 可以匹配 `glad` 或 `good`                  |
| ()+     | 多个重复群组的判别                        | `A(xyz)+C` 可以匹配 `AxyzC`、`AxyzxyzC`、`AxyzxyzxyzC` 等 |
